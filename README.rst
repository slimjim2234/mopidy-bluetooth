Mopidy extension for bluetooth capability


Installation
============

Install by running::

    pip install Mopidy-Bluetooth

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============

Before starting Mopidy, you must add configuration for
Mopidy-Bluetooth to your Mopidy configuration file::

    [bluetooth]
